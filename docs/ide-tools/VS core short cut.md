# Visual studio Core 快捷
### 行操作
* alt+上 或 alt+下 快速移動一整行
* shift+alt+down 複製一行
* ctrl+shift+k 刪除一行

### 快速選擇
* ctrl+F2 選取所有相同的名稱
* alt+ mouse click 多重選擇

### 其他
* F8 依序查看警告訊息
* alt+F12 查看定義
* crtl+k, crtl+0 摺疊
* crtl+n 分割視窗
