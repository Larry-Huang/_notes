# Visual studio Core 快捷
### 行操作
* ALT + 上 或 ALT + 下 快速移動一整行
* SHIFT + ALT + down 複製一行
* CTRL+SHIFT+k 刪除一行
* CTRL + X 刪除目前行
* SHIFT + ALT + ↓ 複製此行到下一行

### 快速選擇
* CTRL + F2 選取所有相同的名稱
* ALT + mouse click 多重選擇
* CTRL + SHIFT + 左右方向鍵  一次移動多選取一個單詞
* SHIFT + 上下方向鍵 上下移動選取區塊
* 
### 搜尋
* CTRL+SHIFT+F 在檔案中尋找

### 其他
* F8 依序查看警告訊息
* ALT + F12 查看定義
* CTRL + k, CTRL+0 摺疊
* CTRL + n 分割視窗
